# Installation


## Install the Library


<CodeGroup>

  ```bash {{ title: 'npm' }}
  npm install @latentcat/uvcanvas-react
  ```

  ```bash {{ title: 'yarn' }}
  yarn add @latentcat/uvcanvas-react
  ```

  ```bash {{ title: 'pnpm' }}
  pnpm install @latentcat/uvcanvas-react
  ```

</CodeGroup>


## Use It!


```tsx
import { Lumiflex } from "@latentcat/uvcanvas-react"

export default function Home() {
  return (
    <div>
      <Lumiflex />
    </div>
  )
}
```
