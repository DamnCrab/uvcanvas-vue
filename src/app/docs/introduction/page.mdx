# Introduction


相信不少初入 AIGC 的玩家都有这样的体验：b 站视频、知乎文章多到看不过来；对配环境有着来自内心深处的恐惧；每天都被信息的洪流所淹没，新技术、新产品如雨后春笋般涌出，连试一试都要鼓足勇气……

我也有这这样的苦恼，曾经对新技术保持距离、保持敏感的策略失效了，“一觉醒来，时代变了”成了日常，生怕今天不学不看明天就 out 了，也怕今天猛学猛干，明天出的新技术让一切成了沉没成本。毕竟在这个领域，大多数人不能引领时代，但大环境大步向前时，顺势而为也能躺赢。所以，我们发起了这样一个完全面向 AIGC 使用者的文档网站，希望能给读者带来一些帮助。
